# Decisões Arquiteturais (ADRs) — PSCC Team

Registro de decisões estruturais tomadas pelo time. Consulte antes de propor mudanças em padrões existentes.

## Formato

Cada ADR é um arquivo `adr-NNN-titulo.md` com:

- **Contexto**: o problema ou situação que motivou a decisão
- **Decisão**: o que foi decidido
- **Motivo**: por que essa opção foi escolhida
- **Consequências**: o que muda, o que fica restrito
- **Alternativas consideradas**: o que foi descartado e por quê
